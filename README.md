# Healthcare Insurance Prediction

## Project Overview

This Machine Learning project predicts healthcare insurance charges based on factors such as age, BMI, smoking habits, gender, and region.

## Problem Statement

Insurance companies need an accurate way to estimate medical insurance charges. This project uses XGBoost Regression to predict insurance costs with high accuracy.

## Features

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Encoding
* Feature Scaling
* XGBoost Regression Model
* Performance Evaluation

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* XGBoost

## Model Performance

* R² Score: 0.87

## Dataset Features

* Age
* Sex
* BMI
* Children
* Smoker
* Region
